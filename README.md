# House Price Prediction using TensorFlow

## Overview
This project builds a simple regression model using TensorFlow and Keras to predict house prices based on the number of bedrooms.

The model learns the relationship between bedrooms and price using a single Dense layer.

---

## Technologies
- Python
- TensorFlow / Keras
- NumPy

---

## Model Details
- Input: Number of bedrooms (1 feature)
- Architecture: Single Dense layer with 1 unit
- Optimizer: Stochastic Gradient Descent (SGD)
- Loss Function: Mean Squared Error (MSE)
- Training: 500 epochs
